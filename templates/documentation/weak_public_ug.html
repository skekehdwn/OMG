탭 기능

  1. 취약점 우측 상단에 [Windows], [Linux] 탭을 통해 Windows와 Linux 취약점을 볼 수 있습니다.

검색 기능

  1. 검색 기능을 통해 원하는 취약점 항목만을 표출합니다.
  2. 취약점 항목 이름 또는 일자의 키워드를 입력하여 검색이 가능합니다.

상세 페이지 이동

  1. 진단 항목을 클릭할 경우, 해당 항목의 상세 페이지로 이동합니다.

양호 / 취약 툴팁

  1. 취약점의 판단기준 아이콘에 마우스 커서를 이동할 경우, 진단 항목의 양호 및 취약 판단기준이 툴팁으로 표출됩니다.
