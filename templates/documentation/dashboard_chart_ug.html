Bar 차트

• 현재 보유한 자산의 항목별 수를 Bar 차트로 표출합니다. (상위 5개)
• 해당 차트는 다음과 같이 수행합니다.
  1. Bar 위로 커서를 이동할 경우, 각 항목별로 항목명과 수를 툴팁으로 표출합니다.
    각 대역별 서버 수 (상위 5개)
    물리서버 벤더별 수량 (상위 3개)

Line 차트

• 최근 일 단위로 자산의 항목별 수에 변화 추이를 Line 차트로 표출합니다.
• 해당 차트는 다음과 같이 수행합니다.
  1. Line 차트 영역중 일부 기간을 지정(drag)할 경우 지정된 일자의 그래프를 볼 수 있습니다.
  2. 전체 영역을 다시 지정(drag)하면 원래 상태로 돌아갑니다.
    서버 총 수량 추이 그래프 (최근 30일)

원형 차트

•전체 자산에서 각 항목별 수의 비율을 Pie 또는 Donut 차트로 표출합니다. (상위 3개)
•해당 차트는 다음과 같이 수행합니다.
  1. 차트 영역에 커서를 이동할 경우, 각 항목별로 항목명과 수를 툴팁으로 표출합니다.
  2. 차트 영역을 클릭할 경우, 클릭한 영역을 확대하여 강조합니다.

게이지 차트

• 현재 보유한 자산의 현재 상태를 Gauge 차트로 표출합니다.
• 차트 내부에는 비율 및 수량을 표시합니다.
• 해당 차트는 다음과 같이 수행합니다.
  1. 좌측 상단 탭을 클릭할 경우, 각 차트의 상태 비율 및 수량이 표출합니다.
  2. 현재 차트의 상태 비율은 95%, 75%, 60% 이상 사용중인 자산의 수를 나타냅니다.

레이더 차트

• 감지된 유휴 자산 판별 케이스를 7가지로 나누어 연관관계를 Radar 차트로 표출합니다.
• 감지된 알람 케이스는 아래와 같습니다.
  • No Login History : 마지막 로그인 일자 6개월 초과
  • Drive Size No Change : 드라이브 사용량 변화가 없는 자산
  • Listen Port No Change : Listen 상태 포트 수 변화가 없는 자산
  • Established Port No Change : Established된 포트 수 변화가 없는 자산
  • Ram Usage Exceeded : RAM 사용량이 60% 초과
  • Cpu Usage Exceeded : Cpu 사용량이 60% 초과
  • Running Process Exceeded : 실행 중인 프로세스가 100개 이상
• 해당 차트는 다음과 같이 수행합니다.
  1. 레이더 차트 하단의 레전드 위에 커서를 이동하거나 클릭할 경우 그룹을 활성화 또는 비활성화 함으로 원하는 그룹의 연관관계만 볼 수 있습니다.

지도 차트

• 감지된 장애 징후들이 지도에서 알람으로 표출합니다.
• 해당 차트는 다음과 같이 수행합니다.
  1. 지도 차트 우측 상단에 [세계], [국내], [지역] 탭을 통해 지도를 바꿀 수 있습니다.
  2. 지도 차트 우측 상단에 [+], [-] 버튼을 클릭하여 지도를 확대, 축소가 가능합니다.
  3. 확대 시 커서로 잡고 이동함으로 지도의 좌표를 이동시킬 수 있습니다.
  4. 지도 차트 우측 상단에 확장(expand) 아이콘을 통해 화면 전체에 맵 차트를 띄울 수 있습니다.
